AppBar - это панель приложения, которая располагается вверху экрана и содержит содержимое заголовка.
Toolbar - это контейнер для размещения элементов внутри AppBar.
div с классом grow - это пустой блок, который занимает все доступное свободное пространство и помогает выровнять элементы по краям.
Badge - это компонент для отображения значка с количеством уведомлений.
NotificationsIcon - это иконка уведомлений.
AccountCircleIcon - это иконка аккаунта.
